## 🧠 Overview

**OBS Studio** is a powerful application for screen recording, video capture, and professional live streaming.

It supports multiple video and audio sources, scene switching, real-time mixing, and customizable recording settings.

The software offers plugin support, hardware encoding, and advanced controls for creating high-quality broadcasts.

Perfect for streamers, gamers, educators, content creators, and professionals on Windows.
